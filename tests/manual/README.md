# Manual Test Scripts

Manual testing scripts for development and debugging.

## Files

### `test_export_import.go`

Simple export/import test for development.

**Usage:**

```bash
go run tests/manual/test_export_import.go
```

## Purpose

These tests are for:
- Quick manual verification during development
- Debugging specific scenarios
- One-off testing

For automated tests, see `tests/integration/` instead.
