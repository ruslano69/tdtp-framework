## Usage for AI Agents

Instead of reading entire files, search the JSON maps:

```bash
# Find all Export functions
jq '.files[].functions[] | select(.name | startswith("Export"))' codebase_map.json

# Find RabbitMQ related code
jq '.files[].functions[] | select(.name | contains("RabbitMQ"))' codebase_map.json

# Check dependencies of a module
jq '.dependencies[] | select(.module | contains("postgres"))' dependencies.json
```

## Regenerating Analysis

To update the analysis after code changes:

```bash
# Install funcfinder
git clone https://github.com/ruslano69/funcfinder.git /tmp/funcfinder
cd /tmp/funcfinder && ./build.sh

# Run analysis
./funcfinder --dir /path/to/tdtp-framework --all --json > codebase_map.json
./deps /path/to/tdtp-framework -l go -j > dependencies.json
```
